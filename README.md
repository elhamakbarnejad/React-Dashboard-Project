A responsive dashboard built with React and Tailwind CSS.
The project includes user and product tables with pagination and reusable components.

Features:

- Responsive dashboard layout
- Users table with pagination
- Products table with pagination
- Reusable table and pagination components
- Data fetched from external APIs

Technologies:

- React
- React Router
- Tailwind CSS
- React Icons
- Axios (for API requests)

Installation:

- Clone the repository: git clone
- Go to the project folder: cd game-store
- Install dependencies: npm install
- Run the development server: npm run dev

APIs Used:

- https://dummyjson.com/users
- https://dummyjson.com/products
